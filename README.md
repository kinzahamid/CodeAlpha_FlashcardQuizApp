# 📚 Flashcard Quiz App

A beautifully designed, interactive flashcard application built for the **CodeAlpha Internship** (Task 1 - App Development).

## ✨ Features

- ✅ **Flip flashcards** - Click or press Space to reveal answers
- ✅ **Navigate cards** - Previous/Next buttons or arrow keys
- ✅ **Add flashcards** - Create your own study cards
- ✅ **Edit flashcards** - Modify existing cards anytime
- ✅ **Delete flashcards** - Remove cards you don't need
- ✅ **Mark as Known** - Track your mastered cards
- ✅ **Progress tracking** - Visual progress bar and stats
- ✅ **Categories & Difficulty** - Organize cards by subject
- ✅ **Import/Export** - Backup your data as JSON
- ✅ **Keyboard shortcuts** - Space, ←, →, K, Delete
- ✅ **Mobile responsive** - Works on phones with swipe support
- ✅ **Persistent storage** - Cards saved in browser LocalStorage

## 🎮 Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `Space` | Flip the card |
| `←` | Previous card |
| `→` | Next card |
| `K` | Mark as known/mastered |
| `Delete` | Delete current card |

## 🚀 How to Use

1. Open `index.html` in any modern browser
2. Add your first flashcard using the form
3. Study by flipping cards and marking known ones
4. Track your progress with the statistics dashboard
